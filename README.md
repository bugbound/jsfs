# jsfs - a web based javascript filesystem
Explore the extraordinary world of a websites javascript, via a basic linux filesystem presentation!

## Install
sh ./install-deps.sh

## Run
```sh
python jsfs.py
python jsfs.py --proxy=http://localhost:8080
```
